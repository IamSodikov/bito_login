# bito_login
Job interview task(Unical Solutions Company)

🚀 Bito Login Automation Testing

This repository contains automated tests for the Bito login functionality, implemented using Behave (BDD) and the Gherkin syntax.

📝 About the Task
This project was completed as part of a technical assignment for a QA Automation Engineer role at Unical Solutions Company.

🛠 Technologies Used
Python

Selenium WebDriver

Behave (BDD testing framework)

Gherkin (feature files)

Page Object Model (POM) design pattern

Project Structure:
            bito_login_test/
            ├── features/
            │   ├── login.feature
            │   ├── steps/
            │   │   └── login_steps.py
            │   └── environment.py
            ├── pages/
            │   ├── base_page.py
            │   ├── locators.py
            │   └── login_page.py
            ├── config.py
            └── .env
✅ How to Run the Tests

To run the tests, execute the following command in the terminal:

"behave --no-capture"

🎯 Test Results
The test cases written for the login functionality pass successfully.
